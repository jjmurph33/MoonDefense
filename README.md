# Moon Defense

Built with the [Usagi Engine](https://usagiengine.com/) for [Usagi Jam 2026](https://itch.io/jam/usagi-jam-2026)

## Controls

- `Up`/`Down`: Choose a satellite
- `Left`/`Right` or `BTN1` (Z Key): Launch
- `BTN2` (X Key): Pause

<br>Make sure to keep the shield up!
